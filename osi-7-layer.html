OSI 7 Layer vs TCP/IP 4 Layer
OSI 7 Layer
L7 Application HTTP, FTP, SMTP
L6 Presentation SSL/TLS, JPEG
L5 Session NetBIOS, RPC
L4 Transport TCP, UDP
L3 Network IP, ICMP, ARP
L2 Data Link Ethernet, PPP
L1 Physical Cable, Hub
TCP/IP 4 Layer
L4 Application HTTP, FTP, DNS, SSH
L3 Transport TCP, UDP
L2 Internet IP, ICMP
L1 Network Access Ethernet, Wi-Fi
계층별 핵심 역할
Application: 사용자와 직접 상호작용하는 프로토콜 (HTTP, SMTP)
Transport: 프로세스 간 통신, 신뢰성 보장 (TCP), 포트 번호
Network/Internet: 논리적 주소(IP), 라우팅으로 목적지까지 경로 결정
Data Link/Network Access: 물리적 주소(MAC), 같은 네트워크 내 전송
